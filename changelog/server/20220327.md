# 2022年3月27日

## 新内容

### 模组

1. 加入了Bed Bugs（修复卡床）
2. 加入了Immersive Posts（沉浸长杆）
3. 加入了Immersive Petroleum（沉浸原油）
4. 加入了Sky Lanterns（孔明灯/天灯）
5. 加入了Industrial Craft 2（工业时代2）
6. 加入了KleeSlabs（更好地破坏半砖）
7. 加入了CoroUtil作为Sky Lanterns的前置
8. 加入了BitHop来优化原版漏斗
9. 加入了Converting Industrial Wires（工业线缆：转化版）

### 方块/物品

1. 加入了来自异界之铲（目前不可合成）
2. 加入了矿泉水，引用后可以补充10格水分+1格饥饿值
3. 加入了矿泉水瓶，矿泉水饮用后剩下，可以通过装矿泉水来再次使用
4. 加入了矿泉水（流体），可以用桶或矿泉水瓶装起，会极其罕见地生成
