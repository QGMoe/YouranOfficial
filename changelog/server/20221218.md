# 2022年12月18日

## 新内容

### 指令

1. 加入了/ti，用于玩家间传送。功能与/ta相反，用于邀请别人传送到自己的位置。
2. 加入了/home和/sethome，/sethome可设置家的位置，/home可快速回家（传送到设定的位置）。如果先前没有设置家，则/home会将你传送到床的位置。
3. 加入了/back，可用于快速回到最后一次传送前的位置或死亡的位置。
